

## 说明
混合模式是指同一个显示容器中的两个显示对象重叠时，重叠区域如何呈现的方式，也就是两个显示对象的重叠区域像素如何混合产生结果像素。 

覆盖混合，表示为"normal"，该显示对象出现在背景前面。显示对象的像素值将覆盖背景的像素值。在显示对象为透明的区域，背景是可见的。


## 设置
通常不设置混合模式，默认就是覆盖混合。设置图片为覆盖模式的代码：    
```img.blendMode = egret.BlendMode.NORMAL;```
